Keyboard Shortcuts

Key Combination Action
return Execute current statement
shift-return Add newline to current code cell, making room for another statement
Cmd-backspace Delete cell (confirmation)
Ctrl-opt-c Copy cell
Cmd-opt-b Change cell type (text/code)
opt-uparrow Focus on previous cell
opt-downarrow Focus on next cell
Cmd-opt-uparrow Move cell up one position
Cmd-opt-downarrow Move cell down one position
/* Open a new text cell
*/ Close the current text cell
